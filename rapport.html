Application web : Rapport de projet

Site web Hearthstone

Introduction

Pour ce projet, nous avons choisi de réaliser un site web en rapport avec un hobby que nous partageons : le jeu vidéo Hearthstone.

Ce site présente le jeu et en explique les règles. Sur la page "Decks", on trouve également des exemples de decks très populaires et qui peuvent servir d'exemples aux joueurs débutants. Enfin, sur la page "Arena draft", l'utilisateur peut s'entraîner à un mode de jeu particulier, où le joueur doit réaliser un deck à partir de cartes lui étant proposées alétoirement avant d'affronter d'autres joueurs ayant réalisé leur deck de la même manière. Nous avons recréé le mécanisme permettant au joueur de confectionner un deck à partir de cartes aléatoires, et une fois que l'utilisateur à fini sa sélection, nous nous basons sur des guides écrits par des joueurs vétérans pour attribuer une note à leurs choix.


Technologies utilisées

  • HTML et CSS servent bien sûr de base pour ce site.
  • Javascript a également beaucoup été utilisé, notament pour la réalisation de la page "Arena Draft"
  • JQuery a aussi été utilisé pour les pages "Arena Draft" et "Decks"
  • Nous avons aussi utilisé Node.js pour créer un web service que l'on pourra interroger pour récupérer les informations qui nous intéressent sur les cartes
  • Finalement, nous avons utilisé AJAX pour interroger une base de donnée JSON contenant les différentes cartes

Nous avons également utilisé la bibliothèque Bootsrap, qui utilise les technologies ci-dessus et permet d'obtenir rapidement un bon rendu visuel

Important : il faut lancer le service web cardstierws.js avec Node.js pour que le site fonctionne correctement


Contenu détaillé des pages

Accueil

Cette page présente très brièvement Hearthstone. Elle contient notament un caroussel présentant quelques captures d'écran du jeu défilant automatiquement ou bien lorsque l'utilisateur clique sur les flèches. Enfin, cette page, comme toutes les autres, contient un logo, une barre de navigation et un pied de page

Règles

Cette page présente plus en détail les méchanismes présents dans Hearthstone. Elle n'est pas très interactive mais peut s'avérer nécessaire pour un visiteur ne connaissant pas du tout le jeu.

Decks

Cette page présente trois types de decks populaires de Hearthstone : le mage freeze, le guerrier contrôle et le chasseur agressif. Lorsque l'utilisateur clique sur un des trois portraits de héros, la liste des cartes du deck ainsi qu'une rapide présentation apparaissent en bas de la page.

Arena Draft

Comme expliqué plus haut, cette page simule l'un des modes de jeu et permet à l'utilisateur de s'entraîner à créer un deck d'arène, c'est à dire un deck constitué à partir de cartes lui étant proposées aléatoirement

Cette page demande dans un premier temps à l'utilisateur de sélectionner un héros, puis la page va sélectionner 3 cartes dans une base de données JSON, les présenter à l'utilisateur et lui demander d'en choisir une avant de lui présenter 3 nouvelles cartes etc... jusqu'à ce que 30 cartes aient été choisies. Comme dans le mode d'arène de Hearthstone, un filtre est appliqué de manière à ne présenter que les cartes normalement accessibles au héros sélectionné et un autre filtre assure que les 3 cartes présentées auront toujours le même niveau de rarerté (3 comunes, 3 rares, 3 épiques ou 3 légendaires)
