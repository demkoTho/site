Application web : Rapport de projet

Site web Hearthstone

Introduction

Pour ce projet, nous avons choisi de réaliser un site web en rapport avec un hoby que nous avons tout les deux en commun : le jeux vidéo Hearthstone.

Ce site présente le jeu et en explique les règles. Sur la page "Decks", on trouve également des exemples de decks très populaires et qui peuvent servir d'exemple aux joueurs débutants. Enfin, sur la page "Arena draft", l'uttilisateur peut s'entrainer à un mode de jeu particulier, où le joueur doit réaliser un deck à partir de cartes lui étant proposer alétoirement avant d'affronter d'autres joueurs ayant réaliser leur deck de la même manière. Nous avons recréé le mécanisme permettant au joueur de confectioner un deck à partir de cartes aléatoires et une fois que l'uttilisateur à fini sa selection, nous nous basons sur des guides écrits par des joueurs vétéran pour attribuer une note à leurs choix.


Technologies uttilisées

  • HTML et CSS servent bien sur de base pour ce site.
  • Javascript à également beaucoup été uttilisé, notament pour la réalisation de la page "Arena Draft"
  • JQuery a aussi été uttilisé pour les pages "Arena Draft" et "Decks"
  • Finalement, nous avons uttilisé ajax pour interroger une base de donnée Json contenant les différentes cartes

Nous avons également uttilisé la biblithèque Bootsrap, qui uttilise les technologies ci-dessus et permet d'obtenir rapidement un bon rendu visuel


Contenu detaillé des pages

Accueil

Cette page présente très briévement Hearthstone. Elle contient notament un caroussel présentant quelques captures d'écran du jeu défilant automatiquement ou bien lorsque l'uttilisateur clique sur les flèches. Enfin, cette page, comme toutes les autres, contient un logo, une barre de navigation et un pied de page

Règles

Cette page présente plus en détail les mechanismes présent dans Hearthstone. Elle n'est pas très interractive mais peut s'averer nescessaire pour un visiteur ne conaissant pas du tout le jeu.

Decks

Cette page présente trois types de decks populaires de Hearthstone : le mage freeze, le guerrier controle et le chasseur agressif. Lorsque l'uttilisateur clique sur un des trois portraits de héro, la liste des cartes du decks ainsi qu'une rapide présentation apparaissent en bas de la page.

Arena Draft

Comme expliqué plus haut, cette page recréé une partie du jeu et permet à l'uttilisateur de s'entrainer à créer un deck d'arène, c'est à dire un deck constitué à partir de cartes lui étant proposé aléatoirement

Cette page demande dans un premier temps à l'uttilisateur de sélectioner un héro, puis la page va sélectioner 3 cartes dans une base de donnée Json, les présenter à l'uttilisateur et lui demander d'en choisir une avant de lui presenter 3 nouvelles cartes etc jusqu'à ce que 30 cartes aient été choisies. Comme dans le mode d'arène de Hearthstone, un filtre est appliqué de manièreà ne présenter que les cartes normalement accessibles au héro sélectioner et un autre filtre assure que les 3 cartes présentées auront toujours le même niveau de rarerté (3 comunes, 3 rares, 3 épiques ou 3 légendaires)/
